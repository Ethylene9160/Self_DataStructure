# 迭代与递归

斐波那契数列：1，1，2，3，5，8，...

假设有个获取第n位斐波那契数的方法`getFNum(n)`, 获取第40个斐波那契数：

* 迭代法

```c
int getFNum(int n) {
    if (n < 0)return 0;
    if (n == 1) return 1;
    if (n == 2) return 2;
    int i;
    int*a = new int[n];
    a[0] = 1;
    a[1] = 1;
    for (i = 2; i < n; i++) {
        a[i] = a[i - 1] + a[i - 2];
    }
    int m = a[n - 1];
    delete[] a;
    return m;
}
```

output: `102334155`



```c
int getFNum(int n){
    if(n == 1) return 1;
    if(n == 2) return 1;
    return getFNum(n-1)+getFNum(n-2);
}
```

output: `102334155`

后者明显比前者慢。
